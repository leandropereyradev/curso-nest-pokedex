# Run in development

1. Clone the repository
2. Run `npm install` inside the project folder
3. Have Nest CLI installed `npm i -g @nestjs/cli`
4. Run the Database `docker-compose up -d`

## Used Stack

- Nest
- Docker
- MongoDB
